# UpCar — Drive Your Dream 🚗

O ecossistema completo do seu veículo. Da caderneta digital ao maior app automotivo do Brasil.

## Stack

- **Next.js 15** — Framework React
- **TypeScript** — Tipagem estática
- **Tailwind CSS** — Estilização
- **Firebase** — Backend e banco de dados
- **Framer Motion** — Animações

## Rodando localmente

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000)

## Configuração Firebase

1. Copie `.env.local.example` para `.env.local`
2. Preencha com suas credenciais do Firebase
3. No Firestore, a coleção `waitlist` será criada automaticamente

## Deploy

Recomendamos a **Vercel** para deploy:

1. Conecte o repositório GitHub na Vercel
2. Configure as variáveis de ambiente
3. Deploy automático a cada push!
